MCP Server

MCP Server for Coding Agents.

AgentML exposes project contracts, allowed paths, validation commands, diff audits, briefs, and skills through an MCP server.

MCP TOOLS
$agentml mcp
Starting MCP server on stdio...
Available tools:
- get_project_contract
- get_agent_brief
- get_allowed_paths
- get_validation_commands
- audit_diff
- validate_contract
- validate_skill
- list_skills
- inspect_skill
- match_skills
- get_skill_guidance

MCP tools do not execute arbitrary shell commands, block path traversal, restrict generated writes, and do not allow skills to override AGENT.agent.

Native Syntax Preview

Native AgentML syntax (experimental).

AgentML is developing its own native block-based syntax on the dev branch. YAML-compatible contracts remain supported and are the default in the current public release.

AGENT.agent (Native Syntax)
$agent "agentml" {
version "0.4.0"
description "AI-native project contract system"
permissions {
write: ["src/**", "tests/**", "docs/**"]
forbidden: [".env", ".git/**", "target/**"]
}
validation {
command: "cargo test"
}
}
